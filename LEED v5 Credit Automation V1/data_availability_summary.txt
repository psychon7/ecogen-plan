DATA SOURCES BY CATEGORY
================================================================================

1. GOVERNMENT APIs & DATABASES
------------------------------

Climate & Weather:
• NOAA Climate Data API (https://www.ncei.noaa.gov/access/services/data/v1)
  - Historical and current weather data
  - Temperature, precipitation, extreme events
  - Free access with API token

• NWS API (https://api.weather.gov)
  - Real-time weather forecasts and alerts
  - JSON-based REST API
  - Free public access

• NOAA Atlas 14 Precipitation Data (https://hdsc.nws.noaa.gov/hdsc/pfds/)
  - Rainfall frequency and depth data
  - Stormwater design calculations

Hazard & Environmental Data:
• FEMA NFHL Web Services (https://hazards.fema.gov/arcgis/rest/services/public/NFHL/MapServer)
  - Flood hazard zone maps
  - GIS web services (WMS, WFS)
  - Free public access

• USGS 3DEP Elevation Service (https://apps.nationalmap.gov/epqs/)
  - Digital elevation models
  - Slope analysis data
  - Point query service available

• National Wetlands Inventory (https://www.fws.gov/program/national-wetlands-inventory)
  - Wetland location and classification
  - GIS data layers

• USFWS Critical Habitat Portal (https://ecos.fws.gov/ecp/)
  - Endangered species habitat data
  - GIS-compatible formats

Demographic & Social Data:
• US Census Bureau API (https://www.census.gov/data/developers.html)
  - American Community Survey (ACS) data
  - Block group level demographics
  - Free API access

• EPA EJScreen (https://www.epa.gov/ejscreen)
  - Environmental justice screening data
  - 12 environmental + 6 demographic indicators
  - Feature service available

Energy & Grid Data:
• EPA eGRID (https://www.epa.gov/egrid)
  - Grid emission factors by region
  - CO2, CH4, N2O emission rates
  - Annual data releases

• NREL PVWatts API (https://developer.nrel.gov/docs/solar/pvwatts/)
  - Solar energy production estimates
  - Location-specific solar resource data
  - Free API with registration

• NREL National Solar Radiation Database (NSRDB)
  - Solar resource data
  - Weather data for solar modeling

Soil & Agriculture:
• USDA NRCS Soil Surveys (https://websoilsurvey.nrcs.usda.gov/)
  - Soil properties and infiltration rates
  - SSURGO database

• USDA PLANTS Database (https://plants.usda.gov/)
  - Native plant species information
  - Invasive species data

Air Quality:
• EPA AirNow API (https://www.airnow.gov/)
  - Current air quality index
  - PM2.5, ozone, other pollutants

• EPA SNAP Program (https://www.epa.gov/snap)
  - Approved refrigerants
  - GWP values and phase-out schedules

2. COMMERCIAL DATABASES & TOOLS
-------------------------------

Embodied Carbon & LCA:
• EC3 Database API (https://www.buildingtransparency.org/)
  - 20,000+ Environmental Product Declarations
  - Construction material carbon data
  - API access available (free for research)

• Tally LCA Tool (https://kierantimberlake.com/work/tally/)
  - Revit-integrated LCA tool
  - Material database with GWP values
  - Commercial license required

• One Click LCA (https://www.oneclicklca.com/)
  - Building LCA software
  - Extensive material database
  - Cloud-based with API

Product Certifications:
• EPD Registry (https://www.theepdregistry.com/)
  - Global EPD database
  - Construction products
  - Free search, API available

• HPD Repository API (https://www.hpd-collaborative.org/)
  - Health Product Declarations
  - Material health data
  - API for data partners

• GREENGUARD Database (https://www.greenguard.org/)
  - Low-emitting product certifications
  - Indoor air quality certified products

• FloorScore Database (https://www.rfci.com/)
  - Flooring product emissions certifications
  - CDPH Standard Method compliance

• Cool Roof Rating Council (https://coolroofs.org/)
  - Solar Reflectance Index (SRI) values
  - Roofing product ratings

Building Energy:
• ENERGY STAR Databases (https://www.energystar.gov/)
  - Product efficiency ratings
  - Portfolio Manager API
  - EVSE registry

• AHRI Directory (https://www.ahrinet.org/)
  - HVAC equipment efficiency ratings
  - Certified product database

• ASHRAE Standards (90.1, 62.1, 55, 202)
  - Published technical standards
  - Static reference data

RECOMMENDATIONS FOR IMPLEMENTATION
===================================

1. PRIORITY INTEGRATIONS (High Impact, Low Complexity)
   - EPA eGRID: Grid emission factors for all carbon-related credits
   - NOAA Climate API: Weather data for climate resilience credits
   - EC3 Database: Embodied carbon data for materials credits
   - US Census API: Demographic data for equity credits
   - FEMA NFHL: Flood data for site selection credits

2. API ACCESS REQUIREMENTS
   - Most government APIs: Free with registration
   - EC3 API: Free for research, paid for commercial use
   - NREL APIs: Free with API key registration
   - Commercial tools: License fees apply

3. DATA QUALITY ASSURANCE
   - Government sources: High reliability, established QA/QC
   - Commercial databases: Third-party verified data
   - Project-specific data: Requires validation workflows

4. REAL-TIME CAPABILITIES
   - 49% of credits can leverage real-time data
   - Weather, air quality, and energy data available in real-time
   - Design-based credits require static analysis
