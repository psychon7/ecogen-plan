### **Option 1.** Whole building life-cycle assessment

7

### LCA process

Complete the following fields to describe the LCA process.

Timeline

Identify when the initial LCA study was performed.

Project team

Identify who conducted and participated in the LCA study (name(s), company(ies), title(s), and project role(s)).

Software tool

Identify the software tool used for the baseline and proposed buildings in the LCA.

Data set

Identify the data set used to perform the LCA. If using a dataset other than one for the specific project location, describe why the data set was selected as a proxy.

Confirm the data set used for the LCA is compliant with ISO 14044.

### LCA scope

Complete the following fields to describe the LCA scope.

Confirm that the major structure, enclosure, and hardscape materials of the building AND all ancillary structures (such as parking structures or outbuildings within the LEED project boundary) have been included in the LCA.

Confirm that controls, fire detection / alarm system fixtures, elevators, conveying systems, excavation, site development, and parking lots have been excluded from the LCA.

Confirm that the service life used in the LCA is at least 60 years.

System boundaries

Identify the system boundaries of the LCA.

### LCA assumptions

Complete the following fields to describe the LCA assumptions.

Defining the baseline approach

Identify the approach used for defining the baseline building (e.g. early design, existing building, building archetype), or define the approach used.

Confirm that the baseline and proposed buildings in the LCA are of comparable size, function, orientation, and operating energy performance as defined in EAp2: Minimum Energy Efficiency.

Standard design and material selection

Describe why the baseline structural system, enclosure, and hardscape represent typical construction for this project, location, and building type.

Provide supporting documentation, such as industry information, demonstrating typical construction for this project, as described above.Upload documentation

Actual material quantities

Identify the approach used to determine the actual material quantities for the project in the proposed model.

Structural system

Describe the structural system of the baseline building in the LCA. Compare and contrast the structural system of the proposed building in the LCA.

Data sources - structural

Provide the data sources used to determine the embodied carbon emissions intensities for each material in the structural system.Upload documentation

When EPDs were not available as a baseline data source, justify the baselines used, demonstrating that the required hierarchy of data has been followed.

Enclosure

Describe the enclosure of the baseline building in the LCA. Compare and contrast the enclosure of the proposed building in the LCA.

Data sources - enclosure

Provide the data sources used to determine the embodied carbon emissions intensities for each material in the enclosure.Upload documentation

When EPDs were not available as a baseline data source, justify the baselines used, demonstrating that the required hierarchy of data has been followed.

Hardscape

Describe the hardscape of the baseline building in the LCA. Compare and contrast the hardscape of the proposed building in the LCA.

Data sources - hardscape

Provide the data sources used to determine the embodied carbon emissions intensities for each hardscape material.Upload documentation

When EPDs were not available as a baseline data source, justify the baselines used, demonstrating that the required hierarchy of data has been followed.

### Results

Embodied carbon

Enter the project's embodied carbon in kg CO2e.

Embodied carbon reduction

Select the project's reduction in embodied carbon.

Project documents

Provide the relevant project documents, including drawings and specifications that document the type and quantity of structure, enclosure, and hardscape materials actually used on the project.Upload documentation

LCA report and data output files

Reduction strategies
