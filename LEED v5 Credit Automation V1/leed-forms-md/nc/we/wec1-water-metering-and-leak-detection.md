### **Option 1.** Submeters

1

Applicable subsystems

Identify the applicable water subsystems on the project. All applicable subsystems must be metered.

Metered fixtures and fittings calculation

Enter the percentage of total indoor fixtures/fittings that are metered. Must be at least 80%.

Water meter design documentation

Provide contract documents indicating the locations and types of the project's permanent water meters for each applicable subsystem.Upload documentation

Real-time submeter access

Provide contract documents or a narrative demonstrating facility manager and/or tenant(s) will be able to access the submeter data in real-time via local network, BMS, cloud service, app, or online database.Upload documentation

Real-time metering capability

Provide documentation confirming that all submeters are capable of recording data at least hourly.Upload documentation

### Healthcare only

In addition to the requirements above, select at least five subsystems that water meters are installed for. If a healthcare project does not include five of the additional subsystems listed above within the project scope, the project may alternatively submeter all water subsystems that are applicable to the project scope.

### Residential only

Confirm that a permanent water meter is installed for each residential dwelling unit that measures the total potable water use for the unit.

### **Option 2.** Leak detection sensors

1

Applicable water-using systems

Identify the applicable subsystems (irrigation, flush fixtures, makeup water) included in the project.

Leak detection sensor documentation

Provide contract documents indicating the locations and types of the project's permanent water flow meters and/or sensors for each applicable subsystem.Upload documentation

Leak detection infrastructure description

Provide a narrative or documents indicating the leak detection system specifications for when and how an alarm is triggered.Upload documentation

Leak detection for flush fixtures calculation

Enter the percentage of the total flush fixtures that have leak detection. Must be at least 50%.

Leak detection action plan

Provide the project's action plan that addresses how the building manager or tenant will have access to data in real-time and how the building manager and/or tenant(s) will address and remedy any detected leak.Upload documentation
